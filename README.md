# Shinyapp SSB-template v001
Dette er en template for å starte opp en shinyapp med eksisterende layout og design fra konkursar-appen skrevet av Solveig Bjørkholt.

Original-repo (det mangler shapefiles, css, logofil, csv):
https://github.com/statisticsnorway/konkursar

En god del kode for å lage grafer, shapefiles og innhold fra konkursar-appen er strippet ut for å gi ett minimalistisk utgangspunkt, men det kan være vel verdt å sjekke ut kildekoden for konkursar for å se hvordan plots ble skrevet med plotly der, norgeskart laget, eller hvordan filtreringen applikeres på dataene.

Template gjort tilgjengelig av Carl F. Corneil, 2020-12-15, carl.corneil@ssb.no



## Readme fra den orginale konkursar-appen

Code for app: https://statisticsnorway.shinyapps.io/konkursar/

The app is based on data from SSB's API on weekly bankrupcies: https://www.ssb.no/en/statbank/table/12972/

The data on geographical distribution in Norway comes from shapefiles, and these data are needed to run the code. 
If the data for the shapefiles is needed, please send a request to solveig.bjorkholt@ssb.no.
